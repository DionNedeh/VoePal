# VoePal

VoePal is an offline-first PWA companion app for short voice and text emotional check-ins. It uses local recording, editable transcripts, a rule-based emotion engine, and a customizable robot friend without API keys, accounts, or uploads.

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run test
npm run build
```

The production build is configured for GitHub Pages at `/VoePal/`.
